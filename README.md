# romiSIM - Global eSIM Seller Platform

A production-ready eSIM marketplace built with Node.js, React, and PostgreSQL. Sell international data plans with instant QR code delivery, secure Paystack payments, and complete admin analytics.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

✅ **200+ Real eSIM Plans** - Africa, Americas, Europe, Asia-Pacific, Middle East  
✅ **Paystack Integration** - Secure payment processing  
✅ **Email Delivery** - Instant QR code and activation codes  
✅ **Admin Dashboard** - Real-time analytics, order management  
✅ **Responsive Design** - Black & white minimalist UI  
✅ **Production Ready** - Error handling, validation, security  

## Tech Stack

- **Frontend**: React 18, Tailwind CSS, Lucide Icons
- **Backend**: Node.js/Express
- **Database**: PostgreSQL
- **Email**: Nodemailer
- **Payment**: Paystack
- **eSIM Provider**: eSIM Access

## Quick Start

### Prerequisites
